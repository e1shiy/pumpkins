1. Условие задачи
В одном сказочном мире живёт женщина, которая любит пинать тыквы. Каждая тыква обладает определённой прочностью 𝑎𝑖, и за каждую разбитую тыкву она получает очки в зависимости от её редкости:

Обычная тыква (1 ≤ 𝑎𝑖 ≤ 99) — 1 очко,
Редкая тыква (100 ≤ 𝑎𝑖 ≤ 199) — 2 очка,
Эпическая тыква (200 ≤ 𝑎𝑖 ≤ 299) — 3 очка.
Каждую тыкву можно разбить только один раз, и прочность её уменьшает силу ноги женщины на 𝑎𝑖.

Цель: Определить минимальную прочность ноги 𝐻𝑃, необходимую для набора 𝑁 очков.

Входные данные: 
𝑁 — количество очков, которое нужно набрать (1 ≤ 𝑁 ≤ 10^5), 
𝑀 — количество тыкв (1 ≤ 𝑀 ≤ 10^2),
𝑎1, 𝑎2, …, 𝑎𝑀 — массив прочностей тыкв (1 ≤ 𝑎𝑖 ≤ 299).

Выходные данные:
Одно число — минимальное 𝐻𝑃, необходимое для набора 𝑁 очков. Если это невозможно, вывести −1.

Пример 1:
Ввод:
10 5
50 120 70 200 90
Вывод:
430

Пример 2:
Ввод:
15 3
50 120 200
Вывод:
-1
